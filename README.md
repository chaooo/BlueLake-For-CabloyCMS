# BlueLake-For-CabloyCMS
A simple theme for Cabloy-CMS

## Installation

``` bash
$ npm i egg-born-module-cms-themeblog-bluelake
```

## Usage

Changing languange's theme to `cms-themeblog-bluelake`

``` javascript
"language": {
  "default": "en-us",
  "items": "en-us"
},
"themes": {
  "en-us": "cms-themeblog-bluelake"
}
```
